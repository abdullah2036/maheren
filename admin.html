🗔 لوحة تحكم مؤسسة الماهرين

اختر لكل صفحة: المشهد الافتراضي، لونًا صافيًا، أو صورة من جهازك (تُصغَّر تلقائيًا وتُحفظ في المتصفح، وتُعرض دائمًا بملء الشاشة دون تشويه).

🚀 نشر تلقائي إلى GitHub (موصى به)

بدل التصدير اليدوي: زر واحد يرفع config.json إلى مستودعك مباشرة، وGitHub Pages يعيد النشر تلقائيًا خلال دقيقة تقريبًا. تحتاج مرة واحدة إنشاء رمز وصول من GitHub: Settings ← Developer settings ← Fine-grained tokens ← Generate، واختَر مستودع الموقع فقط وصلاحية Contents: Read and write.

الطريقة اليدوية (احتياط)

التغييرات تُحفظ تلقائيًا في متصفحك أنت فقط (localStorage). لتعميمها يدويًا: «تصدير config.json» ثم ضع الملف بجانب index.html في المستودع وارفعه.
